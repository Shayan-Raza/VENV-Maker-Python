## Installation
## Linux
```
git clone git@github.com:Shayan-Raza/VENV-Maker-Python.git
```
```
cd VENV-Maker-Python
```
```
chmod +x venv.sh
```
```
./venv.sh
```
Activating the Virtual Environment
```
source <<venc_name>>/bin/activate
```
